### Beihu-Leaf

> 分布式唯一ID服务，支持REST、RPC调用，供参考学习！

##### 本项目基于Beihu-Boot、Leaf(美团)开发，可独立部署使用，提供REST接口、RPC方式的调用

##### 包含：

| 模块                          | 组件            | 特点描述                                                     |
| ---------------------------- | --------------- | ------------------------------------------------------------ |
| beihu-leaf-core              | Core            | 美团Leaf核心代码                                      |
| beihu-leaf-facade            | Facade          | SofaRpc接口定义                                 |
| beihu-leaf-server            | Server          | 提供REST、RPC接口，获取分布式唯一ID              |
